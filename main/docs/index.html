Golf Tournament (GitHub Pages + AWS)

Static frontend. Set API URL in docs/app.js (API_BASE). Admin actions may require x-admin-key.

1) Create tournament

Add rounds, weights, course Par + Stroke Index, and set team aggregation (average, top X). After create, the editor opens automatically.

2) Edit tournament

Adjust handicaps, round weights, tee times, player codes, and two-man pairings after setup.

3) Enter scores

Players use their unique URL: enter.html?code=XXXX. Hole-by-hole inputs default to 4.

4) Scoreboard

Public URL: scoreboard.html?t=TOURNAMENT_ID. Filter by round and toggle Team/Individual (if applicable).

5) Hole map + yardage

View hole-by-hole outlines and use your current location for yardage to the green with back-tee fallback.
