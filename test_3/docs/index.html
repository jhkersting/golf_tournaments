Golf Tournament (GitHub Pages + AWS)

Static frontend. Set API URL in docs/app.js (API_BASE). Admin actions may require x-admin-key.

1) Create tournament

Add rounds, weights, course Par + Stroke Index, and choose team aggregation (sum/avg, top X).

2) Enter scores

Players use their unique URL: enter.html?code=XXXX. Hole-by-hole inputs default to 4.

3) Scoreboard

Public URL: scoreboard.html?t=TOURNAMENT_ID. Filter by round and toggle Team/Individual (if applicable).
